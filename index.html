PM-backed semaphores, proven with a real local rebuild.

This repo now opens on the project itself instead of a full source mirror. The implementation, review diff, and local runtime proof are separated so each one is easy to inspect.

What ships in the code branch

The clean implementation lives on minix-semaphore: PM call numbers, message ABI, public header, libc wrappers, Process Manager semaphore state, delayed-reply blocking, signal interruption handling, and packaged test95 coverage.

What proves it

The portfolio proof lives on portfolio-minix-semaphore: a committed local rerun, structured result metadata, the focused guest log with Test 95 ok, and the local VirtualBox runner.

Why the default branch is minimal

Reviewers should land on the concept first. Leaving the default branch as a full upstream MINIX mirror made the semaphore work hard to find and easy to miss. So master is now a portfolio landing document, while the source baseline is frozen on minix-source-base to keep the PR diff honest.
